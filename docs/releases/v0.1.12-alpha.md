## 🔧 Technical Fixes

### WebSocket Container Networking
```python
@property
def websocket_url(self):
    if os.environ.get('RUNNING_IN_DOCKER'):
        return f"ws://simplex-client-{self.slug}:{self.websocket_port}"
    return f"ws://localhost:{self.websocket_port}"
```

### ChutneX DA Synchronization

All nodes wait for `DA_COUNT` DAs to register before starting Tor.

### Docker Image Centralization

All images now in `docker/images/` directory.
