# v0.1.8-alpha - Real-Time Infrastructure

**Release Date:** 2025-12-27

---

## 🎯 Highlights

- **Redis Channel Layer** - Production-ready message broker
- **SimplexEventBridge** - Auto-connects to all containers
- **WebSocket Consumers** - Browser live updates
- **Auto-start** - Event Bridge starts with Django

---

## 🔴 Redis Integration
```bash
docker run -d --name simplex-redis \
  -p 6379:6379 redis:7-alpine
```

---

## 🌉 SimplexEventBridge

- Connects to ALL running containers
- Processes `newChatItems` and `chatItemsStatusesUpdated`
- Pushes to browsers via Channel Layer

---

## ⚡ Auto-Start

Old: Two terminals needed
New: Single `python manage.py runserver`
